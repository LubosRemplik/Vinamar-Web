# Stránka „Z letiště" Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Veřejná statická stránka `/z-letiste`, která hostovi vysvětlí cestu z letiště Alicante (ALC) do apartmánu v La Mata.

**Architecture:** Markdown obsah (`web/content/z-letiste.md`) renderovaný server komponentou podle vzoru `okoli` + CTA odkaz na Google Maps trasu + položka v `Nav`.

**Tech Stack:** Next.js App Router, `lib/content` (gray-matter + remark), vitest, Playwright.

## Global Constraints

- Veškerý uživatelský text česky s diakritikou.
- Veřejně jen obecná oblast (La Mata / Torrevieja), **nikdy přesná adresa / GPS / kódy**.
- Žádná nová závislost, žádný mapový API klíč, žádný iframe — mapa je pouze odkaz.
- Texty cen (auto / bus / taxi) jsou orientační.

- [ ] **Step 3: Vytvořit `web/content/z-letiste.md`**

```markdown
---
title: Z letiště
intro: Jak se dostanete z letiště Alicante (ALC) do apartmánu v La Mata.
---
Apartmán leží v La Mata u Torreviejy, zhruba 40–50 minut jízdy od letiště
Alicante-Elche (ALC). Níže najdete tři způsoby, jak cestu zvládnout.

## Půjčení auta

Na letišti má pobočku většina velkých autopůjčoven (přízemí příletové haly).
Cesta po dálnici AP-7 / N-332 trvá zhruba 40–50 minut. U apartmánu lze
zaparkovat. Orientační cena půjčení začíná kolem 25–40 € / den mimo sezónu.

## Veřejná doprava (bus)

Společnost ALSA jezdí z letiště do Torreviejy; jízdenku koupíte u řidiče nebo
online na alsa.es. Cesta trvá přibližně hodinu. Z Torreviejy do La Maty
navazuje místní autobus, případně krátká jízda taxíkem.

## Taxi a transfery

Taxi stanoviště je přímo před příletovou halou. Cesta do La Maty vyjde
orientačně na 50–70 €. Pohodlnou alternativou je předem objednaný soukromý
transfer (např. Welcome Pickups), který na vás počká s cedulkou.

## Mapa a oblast

Přesnou adresu a kód ke vstupu dostanete po potvrzení rezervace. Trasu z
letiště do La Maty si můžete prohlédnout přes tlačítko níže.
```

- [ ] **Step 4: Vytvořit `web/app/z-letiste/page.tsx`**

```tsx
import { readPage, renderMarkdown } from '@/lib/content';

const ROUTE_URL =
  'https://www.google.com/maps/dir/?api=1&origin=Alicante+Airport+ALC&destination=La+Mata,+Torrevieja';

export default async function ZLetiste() {
  const { data, body } = readPage('z-letiste.md');
  const html = await renderMarkdown(body);

  return (
    <main className="max-w-4xl mx-auto px-6 py-10">
      <h1 className="text-3xl mb-2">{data.title as string}</h1>
      <p className="text-ink/80">{data.intro as string}</p>
      <div className="prose mt-8" dangerouslySetInnerHTML={{ __html: html }} />
      <a
        href={ROUTE_URL}
        target="_blank"
        rel="noopener noreferrer"
        className="inline-block mt-6 rounded-lg bg-terracotta px-5 py-3 font-medium text-sand hover:bg-terracotta/90"
      >
        Trasa z letiště ALC do La Mata
      </a>
    </main>
  );
}
```

test('z-letiste page loads with heading and route link', async ({ page }) => {
  await page.goto('/z-letiste');
  await expect(page.getByRole('heading', { level: 1 })).toContainText('Z letiště');

  await page.goto('/');
  await page.getByRole('navigation').getByRole('link', { name: 'Z letiště' }).click();

});
